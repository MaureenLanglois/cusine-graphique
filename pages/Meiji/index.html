Old Television Effect

Logiciels utilisés :

  • Gimp
  • Processing
  • OBS Studio

* Vous pourrez trouver le code à l'adresse internet suivant : www.processing.org/codetveffect.html

** FLV : Flash video; Format de séquence d'images.

Description de la recette :


Tout d'abord, il vous faut trouver une image que nous allons ensuite ouvrir sur Gimp pour modifier sa taille au format 500x500 pixel. Une fois l'image redimensionnée on ouvre Processing et on y insert le code* et on suit les consigne des annotations présente dans le code. Un fois que c'est fait, à l'aide d'OBS Studio Vous allez faire une capture vidéo de la, ou des fenêtres que vous avez créer pour pouvoir les enregistrer au format .flv** et ensuite les convertir en format vidéo.
